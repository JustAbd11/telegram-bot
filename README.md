# 🤖 بوت التحميل - Download Bot

بوت تيليجرام لتحميل مقاطع السوشل ميديا بأعلى جودة.

---

## 📁 هيكل الملفات

```
bot/
├── main.py              # نقطة الدخول الرئيسية
├── config.py            # الإعدادات (التوكن، آيدي الأدمن)
├── database.py          # قاعدة البيانات SQLite
├── keyboards.py         # أزرار البوت
├── requirements.txt     # المكتبات المطلوبة
└── handlers/
    ├── __init__.py
    ├── user.py          # أوامر المستخدمين
    ├── admin.py         # لوحة الأدمن
    └── download.py      # منطق التحميل
```

---

## ⚙️ طريقة التشغيل

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. تشغيل البوت
```bash
python main.py
```

---

## 🔧 الإعدادات (config.py)

```python
BOT_TOKEN = "توكن البوت من BotFather"
ADMIN_ID   = 123456789   # الآيدي الرقمي للأدمن
```

## 🛠 ميزات البوت

### للمستخدم:
- ✅ تحميل من: تيك توك، انستا، إكس، سناب، جاكو
- 🎯 أرسل الرابط مباشرة — يتحمل تلقائياً
- 📲 واجهة أزرار سهلة

### للأدمن `/admin`:
| الميزة | الوصف |
|--------|-------|
| 📊 الإحصائيات | عدد الأعضاء الكلي واليومي |
| 📢 الإذاعة | إرسال رسالة لكل المستخدمين مع خيار التثبيت |
| 🔔 اشتراك إجباري | إضافة/حذف قنوات مع إحصائيات |
| 🚫 الحظر | حظر/رفع حظر المستخدمين |
| 🆕 إشعارات | إشعار فوري عند دخول أي مستخدم جديد |

## 📌 ملاحظات

- يجب أن يكون البوت **أدمن** في القناة لتفعيل الاشتراك الإجباري
- الحد الأقصى لحجم الفيديو المرسل: **50MB** (حد تيليجرام)
- قاعدة البيانات: `bot_data.db` (تُنشأ تلقائياً)
